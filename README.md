# 📊 基金日报 (Fund Daily Report)

这是一个轻量级的基金实时估值查看页面，基于 HTML/JS 开发，无需后端服务器，直接通过浏览器访问。

## ✨ 功能特点

*   **实时估值**：优先使用天天基金盘中估值，每 30 秒刷新。
*   **持仓参考**：指定主动权益基金没有盘中估值时，根据最新披露持仓和股票行情计算参考幅度，并显示持仓覆盖率与信心等级。
*   **误差校验**：正式净值更新后，自动核对上一交易日的持仓参考值，在本机逐只基金记录近期误差并校准后续结果。
*   **持仓盈亏**：支持录入持仓份额，自动计算当日预估或参考收益。
*   **自选管理**：
    *   支持添加、删除关注的基金。
    *   数据保存在本地浏览器缓存 (localStorage) 中。
    *   **配置导出/导入**：生成配置代码，方便在不同设备间同步自选基金和持仓信息。
*   **直观展示**：
    *   实时涨跌幅颜色区分（红涨绿跌）。
    *   领涨/领跌板块高亮显示。
    *   自动统计上涨/下跌家数。
    *   显示今日预估总收益。
*   **移动端友好**：响应式设计，完美适配手机屏幕。

## 🚀 快速开始

### 方式一：直接使用
直接访问部署好的页面：[https://oahnait43.github.io/fund-pages/](https://oahnait43.github.io/fund-pages/)

### 方式二：本地运行
1.  克隆本项目到本地：
    ```bash
    git clone https://github.com/oahnait43/fund-pages.git
    ```
2.  直接打开 `index.html` 可以使用官方盘中估值；持仓参考依赖 `/api/holding-estimates`，需通过 Cloudflare Pages Functions 或兼容的本地开发服务器运行。

### Cloudflare Pages 部署

仓库中的 `functions/api/holding-estimates.js` 会由 Cloudflare Pages 自动部署为同源函数。部署项目时请以仓库根目录作为构建输出目录；纯静态托管不会提供持仓参考接口。

## 🛠️ 使用说明

1.  **添加基金**：点击右上角的 `⚙️` 设置图标，输入基金代码（如 `159361`），点击“添加”。
2.  **录入持仓**：在设置模式下，点击基金卡片上的“持仓: 0”按钮，输入你的持有份额，即可看到当日预估收益。
3.  **删除基金**：在设置模式下，点击基金卡片右侧的“删除”按钮即可移除。
4.  **备份/恢复**：
    *   **导出**：点击“导出配置”，配置代码会自动复制到剪贴板。
    *   **导入**：点击“导入配置”，粘贴配置代码即可恢复之前的数据。
5.  **刷新数据**：点击右上角的“刷新”按钮，或者等待页面自动刷新（每30秒）。

## 📝 数据来源

官方净值、盘中估值、基金持仓和证券行情来自 [天天基金网](https://fund.eastmoney.com/) 及东方财富公开接口。官方盘中估值始终优先；持仓参考不是基金公司公布的净值，未披露仓位按 0% 贡献处理，不会把前十大持仓归一化到 100%。详细口径见 [ESTIMATION.md](ESTIMATION.md)。
