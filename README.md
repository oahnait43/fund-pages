# 📊 基金日报 (Fund Daily Report)

这是一个轻量级的基金实时估值查看页面，基于 HTML/JS 开发，无需后端服务器，直接通过浏览器访问。

## ✨ 功能特点

*   **实时估值**：通过 JSONP 直接调用天天基金接口，支持毫秒级数据刷新。
*   **自选管理**：支持添加、删除关注的基金，数据保存在本地浏览器缓存 (localStorage) 中。
*   **直观展示**：
    *   实时涨跌幅颜色区分（红涨绿跌）。
    *   领涨/领跌板块高亮显示。
    *   自动统计上涨/下跌家数。
*   **移动端友好**：响应式设计，完美适配手机屏幕。

## 🚀 快速开始

### 方式一：直接使用
直接访问部署好的页面：[https://oahnait43.github.io/fund-pages/](https://oahnait43.github.io/fund-pages/) (假设已开启 GitHub Pages)

### 方式二：本地运行
1.  克隆本项目到本地：
    ```bash
    git clone https://github.com/oahnait43/fund-pages.git
    ```
2.  直接用浏览器打开 `index.html` 文件即可。

## 🛠️ 使用说明

1.  **添加基金**：点击右上角的 `⚙️` 设置图标，输入基金代码（如 `002910`），点击“添加”。
2.  **删除基金**：在设置模式下，点击基金卡片右侧的 `✕` 按钮即可删除。
3.  **刷新数据**：点击右上角的“刷新”按钮，或者等待页面自动刷新（每30秒）。

## 📝 数据来源

数据来源于 [天天基金网](http://fund.eastmoney.com/) 提供的公开接口。
